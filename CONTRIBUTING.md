# Contributing to AI Dispatch Portfolio

Thank you for your interest in contributing! This project is designed to showcase production AI dispatch patterns.

## How to Contribute

### Reporting Issues
- Use the [GitHub Issues](https://github.com/Matrix-ops77/ai-dispatch-portfolio/issues) page
- Include a clear description and reproduction steps
- For bugs, include your environment details

### Pull Requests
1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-feature`)
3. Make your changes
4. Ensure tests pass (`npm run verify`)
5. Commit with clear messages
6. Push and open a Pull Request

### Code Style
- Follow ESLint configuration
- Add tests for new features
- Update documentation as needed

## Questions?
Open an issue for discussion before starting major work.
